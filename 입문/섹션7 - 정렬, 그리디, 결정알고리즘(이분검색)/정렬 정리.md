# 정렬 정리

## 🎯 선택 정렬
현재 위치에 들어갈 값을 전체에서 직접 골라서(selection) 한번에 교환하는 정렬

=> 값을 골라 한번만 교환

### 선택 정렬 특징
- 매 단계마다 하나의 값을 선택한다.
  - 아직 정렬되지 않은 구간에서 최솟값(또는 최댓값) 하나만 찾는다.
  - 중간에 교환하지 않는다.
- 선택정렬은 끝날 때까지 위치만 기억한다.
  - 실제 값 교환은 하지않음
  - 최솟값(또는 최댓값)의 인덱스만 추적함
- 한 단계단 교환은 탐색이 끝난 후 단 한번만 교환한다.
- 앞쪽부터 확정 영역을 늘린다.
  - 그래서 j 는 i + 1 이다.

### 선택 정렬 예시
```JavaScript
function solution(arr) {
  for (let i = 0; i < arr.length; i++) {
    let index = i;
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[i] > arr[j]) index = j;
    }
    [arr[i], arr[index]] = [arr[index], arr[i]];
  }
  return arr;
}
```
    1. 현재 기준인 i의 index 를 저장.
    2. i 와 i + 1 을 전체 비교하며 가장 작은 값을 찾아냄
    3. 가장 작은 값의 index 를 저장.
    4. 현재 기준인 i를 가장 작은 값이 위치하도록 서로 교체


## 🫧 버블 정렬
인접한 원소끼리 비교하면서 조건이 맞으면 즉시 교환하고, 한 패스마다 가장 큰 값이 뒤쪽으로 확정되는 정렬이다.

=> 인접 비교 후 즉시 교환

### 버블 정렬 특징
- 항상 인접한 원소끼리만 비교한다.
  - 비교 대상은 오직 j 와 j + 1
  - 떨어진 인덱스끼리 비교하면 버블 정렬이 아니다.
- 비교할때마다 바로 교환한다.
- 한 번의 패스가 끝나면 가장 큰 값 or 작은 값이 제일 뒤로 확정된다.
  - 큰 값이 떠오르듯 올라감. (bubble)
  - 마지막 인덱스에서는 더이상 비교하지 않음
- 바깥 반복문은 확정된 영역의 크기
  - i < n - 1
  - j < n - 1 - i

### 버블 정렬 예시
```JavaScript
function solution(arr) {
  for (let i = 0; i < arr.length - 1; i++) {
    // 불필요한 비교 제거 (i는 확정된 개수)
    for (let j = 0; j < arr.length - 1 - i; j++) {
      if(arr[j]>arr[j+1]){
        // 큰 값이 뒤로 (버블)
        [arr[j], arr[j+1]]=[arr[j+1], arr[j]];
      }
    }
  }
  return arr;
}
```

## 🧩 삽입 정렬
앞쪽 정렬된 영역에 현재 값을 뒤에서부터 비교해 삽입할 위치를 찾고 그 자리에 삽입하는 방식, 위치를 못찾았을 경우 맨 뒤에 삽입한다.

=> 값을 뽑아서 앞 쪽에 끼워넣음.

### 삽입 정렬 특징
- 앞쪽은 항상 정렬된 영역으로 유지한다.
- 현재 값을 뽑아두고 적절한 위치에 삽입한다.
- 교환하는 방식이 아닌 밀기 방식을 사용한다.

### 삽입 정렬 예시
```JavaScript
function solution(arr){
  // i는 삽입 대상의 인덱스
  // i 기준 왼쪽은 이미 정렬된 상태
  for(let i = 1; i<arr.length; i++){
    // 이번에 삽입할 값
    let tmp = arr[i];

    // 뒤에서 앞으로 비교.
    // tmp 보다 크면 한칸씩 뒤로 이동한다.
    let j;
    for(j = i - 1; j >= 0; j--){
      if(arr[j] > tmp) arr[j+1] = arr[j];
      else break;
    }
    // 빈자리에 tmp 삽입
    // 이때 i 위치값 확정.
    arr[j+1] = tmp;
  } 
  return arr;
}
let arr3=[11, 7, 5, 6, 10, 15, 9];
console.log(solution(arr3));
```
    1. [11, 7, 5, 6, 10, 15, 9] 시작
    2. i = 1 일때, tmp는 7 [7, 11, 5, 6, 10, 15, 9]
    3. i = 2 일때, tmp는 5 [5, 7, 11, 6, 10, 15, 9]
